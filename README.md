# bolloquybeauty
# 🌸 Sitio Web - Estética, Cosmetología & Cosmiatría

Landing page optimizada y de alto impacto visual diseñada para profesionales de la salud estética, cuidado facial y especialistas en cejas y pestañas. 

### 🚀 Características clave:
*   **Diseño Premium & Mobile-First:** Adaptado a cualquier dispositivo usando Bootstrap 5.
*   **Secciones Estratégicas:** Organización limpia de tratamientos, ubicación y políticas de atención mediante componentes colapsables.
*   **Conversión Rápida:** Modales interactivos detallados con accesos directos integrados a la API de WhatsApp para reservas inmediatas.
*   **Estética Cuidada:** Paleta de colores suave (Blanco, Off-White y Rosa Pastel) que transmite confianza, higiene y bienestar.

### 🛠️ Tecnologías utilizadas:
HTML5 • CSS3 (Variables globales) • JavaScript • Bootstrap 5 • FontAwesome
